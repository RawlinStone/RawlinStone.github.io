Tweet Report

For this project in Informatics 133 - User Interaction Software, I created a webpage that helped understand a week's worth of Tweets from RunKeeper, a popular running app. Our job was to parse and filter the information and display it accordingly using TypeScript and Javascript. The graph to the left displays what we had to display in the form of popular activities and the average distance people travelled using certain methods on specific days. The search bar image shows how we had to create a text search function and load tweets from Twitter.

Coded using HTML, CSS, TypeScript and JavaScript.
